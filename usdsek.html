Creating '538-ish' plots with matplotlib¶

Long time R and ggplot2 user trying to get my head around matplotlib.

In this project I wanted to try to make an informative (high data-ink to non-data-ink ratio) and visually appealing plot using matplotlib.

Fivethirtyeight are known for their beatuiful data visualizations and I want to make something that resembles their style.

As it turns out, matploblib is actually quite nice to work with for this since you are very much in control of all things.

For this project I will use data on the exchange rate between the US-dollar and the Swedisk Krona.

The data can be downloaded from here:
https://www.kaggle.com/datasets/lsind18/euro-exchange-daily-rates-19992020

The dataset is in a timeseries format consisting of 5699 days, although some have missing values.

For example the Greek drachma has only 520 observations. Likely because Greece joined the EMU ('Eurozone') in 2001.

Most columns are in a string but some are numeric.

SEK / USD¶

USD/SEK currency pair¶

Currency unit prices are often referenced as currency pairs and consists of a base currency and a quote currency.

The base currency is considered the first currency within the currency pair quotation, with the second part of the quotation being the quote currency.

I want to create the currency pair 'USD/SEK' which, some what counterintuitive we get by taking sek / usd.

Since both are denoted in euros here they are already priced in the same quote currency.

In [629]:
fx['usdsek'] = fx['sek'] / fx['usd']

Quarterly returns¶

To make it easier to create a bar plot (less bars) we'll get the quarterly returns.

In [632]:
import numpy as np

def quarterly(df, pricecol, datecol):
    lastdayofquarter = df.groupby(df.date.dt.to_period('Q'))['date'].max()
    df = df[df['date'].isin(lastdayofquarter)]
    prices = np.array(df[pricecol])
    ret =  prices[:-1] / prices[1:] - 1
    df = df.iloc[1:,] # Since no result for first obs.
    df['quarterly_ret'] = ret
    df['yearquarter'] = df['date'].dt.strftime('%Y-%q')
    df = df[['date','quarterly_ret']]
    return df

usdsek_q = quarterly(fx, 'usdsek', 'date')

fig, ax = plt.subplots()
ax.plot(usdsek_q['date'],usdsek_q['quarterly_ret'])
ax.tick_params(bottom=False)

plt.show()
